# Task One — AI Tools Assignment

Three deliverables built using AI assistance (Claude by Anthropic) 
as part of an Introduction to AI Tools course.

---

## Deliverable 1 — YouTube Video to PPT

**File:** `YouTubeification-of-Software.pptx`

A PowerPoint presentation generated by providing an LLM only the 
link to a YouTube video. The AI watched, understood and structured 
the content into a complete slide deck — no manual summarisation 
done by hand.

---

## Deliverable 2 — Snake Game

**File:** `index.html`  
**Live demo:** https://meek-pavlova-f7e979.netlify.app/

A fully working browser-based Snake game built using AI-assisted 
development. Features include neon visual theme, gem collection 
system, daily rewards, customizable snake skins, speed settings, 
sound toggle, obstacle blocks, and mobile swipe support — all 
generated through iterative prompting in a single session.

---

## Deliverable 3 — Prompt Engineering Reflection

**File:** `task#01.docx`

A written reflection covering what was learned from the lab 
session — including how prompting improved through the 
Brainstorm-Iterate loop, which techniques worked best (Think Hard, 
Context Approach), and challenges faced when the LLM hit its limits.

---

## What I Learned

- How to provide context so an LLM gives the desired output in 
  minimum turns
- The Brainstorm-Iterate loop for refining AI output
- Think Hard and Context approaches for better results
- How to recover when an LLM reaches its context limit
